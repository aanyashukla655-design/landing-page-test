# Simple Landing Page (Internship Test Project)

Hey!  
This is a small landing page I made for a web dev internship test.  
The idea was to take a few sections from the provided Figma design and turn them into
a responsive page using only **HTML, CSS, and Vanilla JS** (no frameworks).

I picked 6 sections that I felt made sense together:
- Navbar  
- Hero  
- Features  
- About  
- Testimonials  
- Footer  

I tried to keep the code simple and readable instead of over-engineering it.  
Everything is built from scratch (no Bootstrap, Tailwind, etc.)
